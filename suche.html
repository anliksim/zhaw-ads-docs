Binary Search

  • Suche nach s in sortiertem Array a mit Index l,m,r
  • falls a[m]<sa[m] < sa[m]<s -> m auf l setzen
  • falls a[m]>sa[m] > sa[m]>s -> m auf r setzen
  • falls a[m]==sa[m] == sa[m]==s -> gefunden
  • falls l+1>=rl+1 >= rl+1>=r -> nicht gefunden
